## ✨ Key Features

This project was built from the ground up to reflect ART6B's values of creativity, collaboration, and innovation, utilizing top-tier frontend engineering techniques.

- **Islands Architecture**: Powered by Astro, the site ships **0KB of JavaScript** for static sections while selectively hydrating interactive React components, ensuring perfect Core Web Vitals and SEO.
- **Double-Bezel UI Architecture**: A custom UI pattern utilizing inner cores and outer shells with sophisticated inset shadowing to mimic physical, machined hardware.
- **High-End Motion Choreography**: Custom `cubic-bezier(0.32,0.72,0,1)` physics curves power all transitions, including a dynamic Fluid Island Navigation and parallax scroll effects.
- **Strict Brand Adherence**: Designed meticulously around the ART6B color palette:
  - **Gunmetal** (`#1B262C`) for depth.
  - **Ford Blue** (`#1351D8`) and **Soft Blue** (`#80C4E9`) for structure.
  - **Light Orange** (`#FF7F3E`) for high-impact action highlights.

## 🚀 Getting Started / Installation

To run this project locally, follow these steps:

```bash
# 1. Clone the repository
git clone https://github.com/yourusername/ART6B-WEB.git
cd ART6B-WEB

# 2. Install dependencies
npm install

# 3. Start the development server
npm run dev
```

To build and preview the optimized production bundle:

```bash
# Build the static HTML output
npm run build

# Preview the local production server
npm run preview
```
